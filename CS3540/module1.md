 - **Regression** – output is continuous (e.g., house price).
 - **Reinforcement Learning** (RL) – An agent interacts with an environment, selects actions, receives rewards (or penalties), and learns a policy that maximizes cumulative reward.

### Instance‑Based vs. Model‑Based Learning
 - **Instance‑Based** – Store training examples; predict by comparing a new instance to stored ones (e.g., k‑Nearest Neighbors).
 - **Model‑Based** – Build an abstract model from all training data (e.g., linear regression).
